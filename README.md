## Introduction
This project is a simple banking system application written in C++. It allows users to create accounts, deposit and withdraw money, and check their balance.

## Features
- Create new accounts
- Deposit money
- Withdraw money
- Check account balance



   cd BankingSystem

   g++ -o BankingSystem main.cpp

   ./BankingSystem
